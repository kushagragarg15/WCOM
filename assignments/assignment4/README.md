# Assignment 4: Rician Fading and Jakes Doppler Spectrum Analysis

## Overview
This assignment focuses on simulating and analyzing the Rician Fading Distribution and the Jakes Doppler Spectrum, fundamental concepts for understanding time-varying fading environments in wireless communication channels.

## Learning Objectives
- Understand Rician fading distribution and K-factor significance
- Simulate Rician fading channels with line-of-sight components
- Analyze Jakes Doppler spectrum and its U-shaped characteristics
- Study autocorrelation functions in time-varying channels
- Evaluate the impact of mobility on channel behavior
- Visualize fading statistics and Doppler effects

## Key Concepts
- Rician fading distribution and K-factor
- Line-of-sight (LOS) vs scattered signal components
- Jakes Doppler spectrum model
- Autocorrelation function and channel coherence
- Bessel functions in wireless channel modeling
- Time-varying channel characteristics

## Files Structure
- `rician_jakes_analysis.m` - MATLAB implementation
- `fading_animation.py` - Interactive Python visualization
- `Assignment4_RicianJakes.md` - Complete documentation
- `Assignment4_WordTemplate.md` - Word document template

## Applications
- Mobile communication channel modeling
- Satellite communication with LOS components
- Vehicle-to-vehicle (V2V) communications
